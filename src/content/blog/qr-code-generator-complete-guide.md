## QR Code Best Practices

### Design
- **Keep contrast high** — dark foreground on light background
- **Leave a quiet zone** — white space around the QR code for scanners
- **Don't go too small** — 2cm × 2cm minimum for print
- **Test on multiple devices** — iOS and Android cameras handle QR codes slightly differently

### Usage
- **Always include a short URL** — long URLs create denser, harder-to-scan codes
- **Add a call-to-action** — "Scan for menu" works better than a plain QR code
- **Track with UTM parameters** — add `?utm_source=qr` to measure scan performance
- **Make it dynamic** — use a redirect URL so you can change the destination later
